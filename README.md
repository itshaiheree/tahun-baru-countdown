# Latar belakang pembuatan
Karena besok tahun baru, jadi yaa... bikin web "sederhana" buat hitung mundur tahun baru karena iseng

(Note: Project ini digarap pada 31/12/25)

# Cara pakai
Kunjungi [tahunbaru.mhai.my.id](https://tahunbaru.mhai.my.id)

# Fitur
1. Liat contdown realtime
2. Endless countdown (Setelah lewat tanggal 1 Januari, dia akan otomatis mulai hitung munudur untuk tahun depan)
3. Tambah "Harapan" yang terintegrasi dengan MongoDB (NEW 🆕)

##


# Mau run sendiri? Bisa dong!
Tinggal clone repo ini atau pull repo ini.

Untuk run dev mode, tinggal run
```
npm run dev
```
atau untuk production-build nya:

```
npm run build
npm start
```

Jangan lupa, setting link, username dan password MongoDB mu di enviroment (.env) dengan format:

```
LINK=[link db]
IYAH=[username db]
PASS=[password db]
```
