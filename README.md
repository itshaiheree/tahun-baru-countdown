# Latar belakang pembuatan
Karena besok tahun baru, jadi yaa... bikin web "sederhana" buat hitung mundur tahun baru karena iseng

(Note: Project ini digarap pada 31/12/25)

# Cara pakai
Kunjungi [tahunbaru.mhai.my.id](https://tahunbaru.mhai.my.id)

# Fitur
1. Liat contdown realtime
2. Endless countdown (Setelah lewat tgl 1 Jan dia akan otomatis mulai hitung munudur untuk tahun depan)
3. Tambah "Harapan" yang terintegrasi dengan MongoDB (COMING SOON)
